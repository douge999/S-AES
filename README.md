## 4、多重加密 
### （1）双重加密：将S-AES算法通过双重加密进行扩展，分组长度仍然是16 bits，但密钥长度为32 bits。 
### （2）中间相遇攻击：假设你找到了使用相同密钥的明、密文对(一个或多个)，请尝试使用中间相遇攻击的方法找到正确的密钥Key(K1+K2)。 
### （3）三重加密：使用48bits(K1+K2+K3)的模式进行三重加解密。
